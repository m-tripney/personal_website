To-do List

The ubiquitous to-do list! This little app allows tasks to be added and deleted, of course, but has the addition of deadlines. With a deadline in place, users can choose how they view their tasks:

  • View all tasks
  • View all tasks for the coming week
  • For added stress, view tasks with a missed deadline

Further, this is a little more advanced than a number of 'traditional' tutorial-style to-do lists. I wanted the tasks to persist. In other words, the user should be able to end their session, come back later, and pick up where they left off, tasks intact — enter SQLAlchemy.

Smart-Calc

A 'smart' calculator app, with user-defined variables and postfix notation. Building something to do simple sums would be straightforward. Here, though, the user can assign values to case-sensitive variables, and then use those variables in their calculations. The app checks the calculations for bracket-matching, and warns if things aren't right.

Behind the scenes, the app converts the user's infix notation to postfix notation, requiring stack manipulation and a careful consideration of operator precedence.

Tic-Tac-Toe

A simple implementation, in Python, of the classic game. Users take turns to place noughts and crosses on the game grid. The first one to make three-in-a-row is the winner! Checks are made for valid coordinates and for valid locations — users can't place their token on an already-occupied square.

And, of course, checks are made for game condition — either the game is ongoing; someone has won; or the game is tied. I'm going to refine this with some AI, so the user can play against the computer. Coming soon!
